Ponify! Lock your Mac or else...

curl -s http://ponify.nl/pony.sh | bash

 

Someone didn't lock their Mac? Let them subtly know they should lock their workstation. Paste the above command in a terminal and their desktop background will be changed to a beautiful Pink Pony wallpaper. No terminal? Use the below buttons to tweet/post/plus the Pony love.

About

Simple bash script which uses curl to download this image and based on the OS X version uses applescript to set the wallpaper to that image. View source code (GPLv3)
